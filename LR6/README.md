# Weather Forecast Example Project (Java)

## Docs

[swagger](http://localhost:8080/swagger)

## Base commands

- build: `gradle build`
- run: `gradle bootRun`
- jacoco: `gradle jacocoTestReport` - Java Code Coverage

## Environment variables

Set environment variables in `.env` file placed in root of project before running:

```shell
# .env
OPENWEATHER_BASE_URL=https://api.openweathermap.org/data/2.5
OPENWEATHER_API_KEY=xxxxxxxxxxxxxxxxx
```
